# Flip Deal Analyzer

Real estate flip deal sourcing and analysis tool.

## Features
- Buy box criteria setup
- Deal entry and analysis
- 70% Rule calculator
- Cash-on-cash ROI
- Deal pipeline tracking
- Mobile-responsive

## Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Supabase (auth + database)
- Zillow API

## Getting Started
1. npm install
2. npm run dev
3. Open http://localhost:3000
